- big prompt
   - takes a lot of tokens
   - adaptive reasoning use reasoning tokens on every small step
   - hard to maintain
   - good for scaffolding - mvp
   - guardrails, ,checklist, rules, examples

- medium
   - roles and focused rule sets

- smaller prompts
   - easier to maintain
   - be explicit, for specific task
   - say what it should do, not what is should not
   - less chance for longer reasoning
   - lower risk of instruction drift
   - more resilient to partial prompt loss - like ignoring some part of prompt

- presenter experiment measured with promptfoo
   - adding prefix "Use best practices:" + prompt itself
   - increased score by 14-16 points (%) by promptfoo
   - used model matters - using different amount of tokens, different success rate
        - gpt-4.1-mini - bad, avg 1990 tokens
                       - 35%
                       - 49% with prefix
        - gpt-5.1 - ok-ish, avg 2500 tokens
                  - 71%

        - gpt5-mini - with reasoning
                    - best, but almost double tokens used - avg 4900 tokens
